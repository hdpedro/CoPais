## 1. Framework AARRR (Pirate Metrics)

## 5. North Star Metric

> **"Grupos com ambos os pais ativos semanalmente"**

Esta metrica captura o valor fundamental do Kindar: coparentalidade funcional com comunicacao ativa entre ambos os pais.

**Por que esta metrica:**
- Requer aquisicao de 2 usuarios (nao 1)
- Requer ativacao (ambos fazendo algo, nao so logando)
- Correlaciona com retencao (grupos ativos voltam)
- Correlaciona com revenue (grupos engajados convertem mais)
- Representa valor real para a crianca (ambos pais envolvidos)

**Decomposicao:**
```
Both Parents Active Weekly =
  Total Grupos x
  % com 2+ membros x
  % com ambos logando na semana x
  % com ambos fazendo >= 1 acao
```
